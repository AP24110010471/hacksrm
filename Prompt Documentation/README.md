# Prompt Documentation

## Prompt Template
When engaging with the AI coding assistant, the following template was effectively used to structure requests:

### Template Structure
**[Objective]**
*Briefly describe what you want to achieve.*

**[Context]**
*Provide relevant file names, current state, or specific constraints.*

**[Requirements]**
*   **Feature A**: Describe specific functionality.
*   **Style**: formatting, aesthetic (e.g., "Farmer-Centric", "Clean", "3D").
*   **Behavior**: Auto-start vs Manual, Click-to-activate, etc.

**[Constraints/Preferences]**
*   *e.g., "Use existing CSS variables", "Do not use external frameworks like React".*
